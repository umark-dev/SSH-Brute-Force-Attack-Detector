# SSH Brute Force Attack Detector

## Description
This project analyzes Linux authentication logs to detect SSH brute-force attacks by identifying repeated failed login attempts from the same IP address.

## Features
- Detects multiple failed SSH login attempts
- Identifies brute-force attack patterns
- Logs successful login events
- Generates alert reports

## Technologies
- Python
- Cybersecurity (Blue Team)
- Log Analysis
- SOC Operations

## How to Run
python detector.py

## Output
Alerts saved in alerts.txt
